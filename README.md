# warranty-ticket-saas-ssr (Angular SSR)

## Quick start

```bash
npm i
npm run start
```

## SSR build & serve

```bash
npm run build
npm run serve:ssr
```

## Notes
- This repo is a **starter skeleton** with a domain-first folder structure (features/core/shared) and SSR-safe patterns.
- All components use **separate** `.html` and `.scss` files.
